# Merge Collections with Vitest

## Install | ENG

### 1. Clone the Repository

```bash
git clone https://github.com/srpthemangoeater/vitest-merge.git
cd vitest-merge
```

### 2. Install Node Modules

```bash
npm install
```

## Excute Vitest

```bash
npx vitest run
```
